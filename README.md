# Trader Performance vs Market Sentiment Analysis

## Objective

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and Hyperliquid trader performance.

## Datasets

- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trader Data

## Workflow

1. Data Cleaning
2. Data Preparation
3. Feature Engineering
4. Exploratory Data Analysis
5. Trader Segmentation
6. Strategy Recommendations

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings

- Extreme Greed had the highest average profit per trade.
- Fear periods had the highest trading activity and trade size.
- Frequent traders outperformed infrequent traders.
- Market sentiment influenced trader behavior and performance.

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Open the notebook:

```
Trader_Performance_Analysis.ipynb
```

3. Run all cells.
